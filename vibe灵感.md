* 💻 VibeDash: 专属极客终端看板

  一个基于终端的个人状态聚合看板，告别枯燥的命令行，用极简与赛博朋克美学掌控开发生活。

  ## ✨ 核心功能 (Features)

  ### 🛠 1. 开发者专属状态 (Developer Stats)

  - **多语言代码追踪：** 接入 API，直观展示当日在 Python、Java、JavaScript 等不同技术栈上的时间分配彩色进度条。
  - **GitHub 绿点渲染：** 在终端直接用 ASCII 色块打印出近期的 GitHub 贡献图与连续 Commit 天数。

  ### 📈 2. 个人进度追踪 (Life & Career Tracking)

  - **简历与求职看板：** 管理近期的 Todo List，清晰展示当前简历版本的优化进度。
  - **习惯打卡与本地分析：** 记录每日的专注时长与生活习惯。底层通过 MySQL 存储，使用 SQL 实时查询并生成本周达标率的终端图表。

  ### 🎛 3. 系统与环境监控 (System Metrics)

  - **硬件资源仪表盘：** 实时滚动的 CPU、内存占用率，以及当前的网络上下行速率。
  - **本地服务健康度：** 通过红/绿状态灯监控本地数据库容器或测试服务器的运行状态。

  ### 🌍 4. 信息聚合枢纽 (Information Hub)

  - **极客资讯速览：** 抓取 Hacker News 的 Top 3 标题，或 GitHub Trending 榜单。
  - **本地问候：** 根据当前 JST (日本标准时间) 自动输出专属的早/晚安 ASCII 艺术字问候与当地天气信息。

  ### 💰 5. 财务与订阅监控 (Finance & Subscriptions)

  - **“吸血”订阅雷达：** 集中管理服务器、各类 API、流媒体等周期性扣费服务。根据距离下次扣费的天数，呈现安全（绿）、警告（黄）或危险（红灯闪烁）的赛博朋克警示色，告别忘记退订的烦恼。
  - **本地化汇率折算：** 调用免费 API，实时获取汇率波动，将多币种（USD/CNY）的订阅费用自动折算为日元 (JPY) 支出预估，在看板底部直观展示。

  ## 🧰 技术栈 (Tech Stack)

  - **核心语言：** Python 3.x
  - **终端 UI 组件：** Rich (负责色彩、图表与排版) + Textual (负责终端交互与布局)
  - **数据存储：** MySQL (本地结构化存储，统一管理所有看板状态、订阅数据与历史记录)
